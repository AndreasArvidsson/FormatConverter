# CsvToJson
Convert a CSV file to a JSON file

## CLI arguments
**Default delimiter ";"**    
```java -jar CsvToJson.jar path/to/input.csv path/to/output.json```    
**User given delimiter**    
```java -jar CsvToJson.jar path/to/input.csv path/to/output.json delimiter```

## CSV Format
```
a;b;c;d
e;f;g;h
i;j;k;l
m;n;o;p
```
